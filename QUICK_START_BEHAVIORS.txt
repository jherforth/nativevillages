SEEING IT IN ACTION:
--------------------
1. Find or build a village (houses with marker blocks like barrels/shrines)
2. Spawn villagers using eggs or wait for natural spawning
3. Fast-forward to night: /time 0
4. Watch villagers walk home and sleep (lay down animation)
5. Fast-forward to day: /time 6000
6. Watch villagers wake up and wander around
7. Spawn multiple villagers near each other - they'll socialize (particles!)
8. Feed one villager with bread, then watch them share food later


TIME OF DAY GUIDE:
------------------
/time 0      = Midnight (villagers sleeping)
/time 6000   = Dawn (villagers wake up)
/time 12000  = Noon (peak activity time)
/time 18000  = Dusk (villagers heading home)
/time 23900  = Almost midnight (villagers should be asleep)

TROUBLESHOOTING:
----------------
Q: Villagers aren't sleeping
A: Make sure it's actually night time (/time 0). Check if villagers have
   a house assigned - newly spawned ones should automatically link to
   nearby houses.

Q: Doors aren't opening
A: Ensure you're using standard doors with group:door. Custom door mods
   might need to be registered properly.

Q: No social interactions
A: Make sure you have 2+ villagers within 5 blocks of each other. Social
   interactions are random (5% chance per second), so be patient!

Q: Villager stuck far from home
A: They'll auto-teleport back after 5 minutes. You can speed this up by
   changing stuck_teleport_threshold in villager_behaviors.lua.


CONFIGURATION:
--------------
Edit: villager_behaviors.lua (lines 10-22)

Key settings:
- home_radius = 20              (how far villagers wander)
- sleep_radius = 3              (distance to trigger sleep)
- social_interaction_cooldown = 45   (seconds between socials)
- food_share_cooldown = 60      (seconds between food shares)
